# Twitch Overlays

This repository hosts Twitch overlays.

## Available Overlays

- **Headpat Overlay**: [Launch](https://mavscriptblu.github.io/TwitchOverlays/MavScript_Overlays/headPat/headpat.html)
- **Stitch Overlay**: [Launch](https://mavscriptblu.github.io/TwitchOverlays/MavScript_Overlays/stitchPat/index.html)

## Setup

To use these as browser sources in OBS:
1. Copy the "Launch" link above.
2. In OBS, add a new **Browser Source**.
3. Paste the URL.
4. Set the dimensions (usually 1920x1080).
